# Nilai mana yang ditampilkan perulangan while?

Untuk setiap iterasi, tulis nilai mana yang dikeluarkan dan bandingkan dengan solusinya.

Kedua perulangan `alert` nilai yang sama, atau tidak?

1. Bentuk prefix `++i`:

    ```js
    let i = 0;
    while (++i < 5) alert( i );
    ```
2. Bentuk postfix `i++`

    ```js
    let i = 0;
    while (i++ < 5) alert( i );
    ```
