# Count properties

Tulis fungsi `count(obj)` yang mengembalikan jumlah properti di dalam objek:

```js
let user = {
  name: 'John',
  age: 30
};

alert( count(user) ); // 2
```

Coba buat kodenya sependek mungkin.

P.S. Jangan pedulikan properti simbolik, hanya hitung yang "reguler".
